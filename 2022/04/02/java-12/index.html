Oracle JDBC 版本与 JDK 版本的关系如何？
Oracle JDBC 驱动程序始终与每一个新版本中的最新 JDK 版本相兼容。在某些版本中，JDBC 驱动程序支持多个 JDK 版本。请使用下表根据您的首选 JDK 版本选择正确的 JDBC 驱动程序。
